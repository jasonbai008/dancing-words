GSAP文字动画特效

本页面展示了使用GSAP和SplitText插件创建的各种文字动画效果

Authors: DeepSeek、Claude 4、Gemini-2.5 Pro、Jason Bai

效果1：字符淡入滑动

文字如诗，动画如画，科技与艺术的完美结合。
let animationSplitText = SplitText.create(element, { type: "chars, words" });
gsap.from(animationSplitText.chars, {
    duration: 1,
    delay: 0.5,
    x: 40,
    autoAlpha: 0,
    stagger: 0.05,
    scrollTrigger: {
        trigger: element,
        start: "top 85%",
        toggleActions: "play none none reverse"
    }
});

效果2：3D旋转淡入

每一个字符都拥有生命，在空间中优雅舞动。
let animationSplitText = SplitText.create(element, { type: "chars, words" });
gsap.from(animationSplitText.chars, {
    duration: 0.8,
    delay: 0.3,
    y: 50,
    rotationX: -90,
    autoAlpha: 0,
    stagger: 0.03,
    ease: "back.out(1.7)",
    scrollTrigger: { 
        trigger: element, 
        start: "top 85%",
        toggleActions: "play none none reverse"
    },
});

效果3：弹性缩放淡入

弹性动画带来活力，让文字充满动感与趣味。
let animationSplitText = SplitText.create(element, { type: "chars, words" });
gsap.from(animationSplitText.chars, {
    duration: 1,
    scale: 0,
    autoAlpha: 0,
    stagger: 0.1,
    ease: "elastic.out(1, 0.5)",
    scrollTrigger: { 
        trigger: element, 
        start: "top 80%",
        toggleActions: "play none none reverse"
    },
});
